# token-generator

Студенту-недоучке дали пробное задание на испытательном сроке:
- разработать готовый микросервис на 17 Java и Spring, который принимает учетные данные пользователя, выпускает для них JWT и отправляет во внешний API.
- все данные, необходимые для подписи токена, должны находиться в конфиге `.yml`
- все тесты обязательно должны проходить

Студент выполнил задание и сразу же залил на бой свои изменения для целого бизнеса. Но он совсем не следовал [рекомендациям безопасности](https://baseline.appsec.ftc.ru/baselines/common/crypto).
